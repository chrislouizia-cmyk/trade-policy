# Trade Police Admin Premium

This release focuses on the private operating system for the company.

## Included

- Premium Mission Control dashboard
- Dedicated left navigation for HQ
- Owner identity and role context
- Customer activity overview
- Operational priority queue
- Private incident summaries
- Staff directory and granular permission controls
- Role-specific Sales, Compliance, Support and Technical workspaces
- Host-based separation for `admin.tradepolice.com`
- Client-host blocking for internal routes
- Responsive HQ layouts

## Product principle

The customer portal is for trading discipline. The Admin Portal is for running the company. They share infrastructure, but they do not share navigation or access expectations.
